# jmt-bench-code

1. **W&B ログイン**  
   ```bash
   wandb login
   ```
   途中で聞かれる API Key は [W&B のアカウントページ](https://wandb.ai/authorize) から取得できます。

2. **Hugging Face ログイン**  
   ```bash
   huggingface-cli login
   ```
   こちらもトークンは [Hugging Face のアクセス トークンページ](https://huggingface.co/settings/tokens) で発行できます。
